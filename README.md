## Seekland

#### Install and Run
> git clone git@github.com:zlailari/seekland.git

> cd seekland

> npm install

> node server.js

#### Ideas Moving Forward
- Multiple worlds with doors between them
- Each person customizes their space
  - Basic HTML/Markdown (links)

- Random respawn
- Powerups
